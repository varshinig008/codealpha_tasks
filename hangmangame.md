# hangman_game

Welcome to my Hangman Game repository!

## Description

This project is a simple, text-based implementation of the classic word-guessing game. The goal is to guess a hidden word one letter at a time before running out of chances. It's a fun and interactive way to test your vocabulary and deduction skills while enjoying a timeless game.

## Features

- **Interactive Gameplay**: Guess letters one at a time to uncover the hidden word
- **Difficulty Levels**: Play with words of varying difficulty
- **Visual Feedback**: See your progress with underscores revealing correct guesses
- **Limited Attempts**: You have a set number of wrong guesses before game over
- **Score Tracking**: Track your wins and losses
- **Simple Interface**: Easy-to-use command-line interface
- **Replay Option**: Play multiple rounds without restarting

## How to Play

1. **Start the Game**: Run the game to begin a new round
2. **Guess Letters**: Enter one letter at a time
3. **Correct Guesses**: If the letter is in the word, it will be revealed
4. **Wrong Guesses**: Each incorrect guess counts against your remaining attempts
5. **Win**: Guess the complete word before your attempts run out
6. **Lose**: If you run out of attempts before guessing the word, the game is over
7. **Continue**: Choose to play again or exit the game

## Getting Started

To get started with this project, clone the repository and run the game:

```bash
git clone https://github.com/varshinig008/hangman_game.git
cd hangman_game
python hangman.py
```

Enjoy the game!
